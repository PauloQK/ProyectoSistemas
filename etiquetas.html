Explicación de las principales etiquetas de HTML


Las etiquetas de HTML son los componentes básicos utilizados para crear y estructurar páginas web.

  1. "DOCTYPE html": Declara el tipo de documento y la versión de HTML que se está utilizando
  2. "html": La etiqueta raíz que envuelve todo el contenido de la página web.
  3. "head": Contiene metadatos sobre el documento, como el título.
  4. "body": Contiene el contenido visible de la página web, como texto, imágenes, videos, etc
  5. "nav": Define un bloque de enlaces de navegación.
  6. "header": Representa la cabecera de la página o sección, generalmente contiene el logo y la navegación principal.
